Configuring vncip

Config files are located in /usr/local/vncip/conf. Within this directory there must be a separate subdirectory for each vncip instance that you run. (For example, you might have a production instance and a test instance.)

The following table summarizes the config files and their contents.

File Section Variable Description
backend.conf sip port The port on which the Voyager SIP2 server is listening
    operator The Voyager operator (staff user) to use in connecting to the SIP2 server
    password The associated password
    location The location code (not location name or display name) at which SIP2 transactions are to occur
  voyager database The Oracle database SID (e.g., VGER)
    item_barcode_format The printf-style format string – which must contain %s – to use in constructing a newly created barcode
  global locations_file The path (relative to this file) of a locations file (see below)
server (none) port The port on which vncip should listen for incoming NCIP1 requests
the locations file each location loan A Boolean value (Y or N) indicating whether requests for items at the location should be honored – one section for each location that needs to be treated specially
